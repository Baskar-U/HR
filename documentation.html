Getting Started with MockVox

Welcome to MockVox API documentation. This guide will help you get started with integrating MockVox's powerful AI models into your applications.

Note: To use the MockVox API, you'll need an API key. You can get one by signing up for an account.

Authentication

All API requests require authentication. To authenticate your requests, include your API key in the headers:

HTTP Request 
GET /api/v1/models HTTP/1.1
Host: api.mockvoxai.com
Authorization: Bearer YOUR_API_KEY

Making Your First API Call

Let's start by generating text with our most powerful model, MockVox-4:

Python 
import requests

api_key = "YOUR_API_KEY"
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

data = {
    "model": "mockvox-4",
    "prompt": "Write a short story about artificial intelligence:",
    "max_tokens": 150
}

response = requests.post(
    "https://api.mockvoxai.com/v1/completions",
    headers=headers,
    json=data
)

print(response.json())
